字符串（Strings）

发表于 2017-05-09 | 分类于 python | 阅读次数

在Python3中，字符串是一个unicode的序列，在Python2中一个byte的序列。字符串的很多特性和元组类似，字符串也是不可变的。

  • '
  • "
  • '''
  • """

Python中单引号和双引号的效果是一样的。

三个单引号或双引号表通常用在字符串为多行的情况下。
