元组

发表于 2017-05-09 | 分类于 python | 阅读次数

元组和列表很像，包括索引查询、查找统计元素操作都很类似。但是元组是不可变的，无法改改元素，所以没有增删改操作。

初始化元组

元组是不可变的，所以初始化后元组内的元素不可改变。

下标/索引查询

  • 元组中的索引，也是从前往后，索引也是从0开始，从后往前，索引是从-1开始。

  • 如果索引超出范围，将引发IndexError异常。

  • 元组中，是无法更改元素的值的，更改会抛出TypeError的异常。

查找/统计元素

index

查找某值的第一个索引，可以指定索引的开始和结束位置。（包含start，不包含stop）

如果值没有找到，则抛出ValueError。

index(…)
T.index(value, [start, [stop]]) -> integer – return first index of value.
Raises ValueError if the value is not present.

count

计数，返回值出现的次数。

count(...)
T.count(value) -> integer – return number of occurrences of value

len函数

len函数可以用在列表、元组、字符串、字典、集合等上面，输出容器中元素的个数。

len(obj, /)
Return the number of items in a container.
