模块化（Modularization)

发表于 2017-05-13 | 分类于 python | 阅读次数

基本概念：

在Python中，模块（modules)、包（packages）和库（librarys）的概念并没有那么清晰，通常我们认为：

  • 一个.py文件就是一个模块（module），模块名就是文件名,例如foo.py，模块名就是foo。
  • 一个目录，包含了__init__.py就是一个包。
  • 当一个包或者若干包，包含一个setup.py就认为是一个可分发的库。
