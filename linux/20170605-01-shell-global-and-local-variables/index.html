01-全局变量和局部变量

发表于 2017-06-05 | 分类于 linux | 阅读次数

全局变量对于shell会话和所有生成的子shell都是可见的。局部变量则只对创建它们的 shell可见。

全局变量（global variables）

用printenv就可以打印全局变量，里面会包括系统生成的全局环境变量和用户自定义的环境变量。

局部变量（local variables）

没有专门的命令查看局部变量，只有一个set命令，会显示当前bash进程设置的所有变量，包括全局和局部。

创建局部变量和全局变量

创建局部变量的方法很简单，就是变量名=值，例如var=10。

把局部变量export之后就是全局变量了。
