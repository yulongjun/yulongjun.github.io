08-ngx_http_fastcgi_module详解

发表于 2017-08-23 | 分类于 linux | 阅读次数

有两种Web架构：

LNMP( Linux+Nginx+MySQL+PHP-FPM+MySQL)：需要FastCGI模块

LNAMP(Nginx+HTTPD+MySQL+PHP_Module)：需要HTTP相关模块

LNMP是使用的php的fpm功能，而不再是一个依赖httpd的库模块。
LNAMP里面，是httpd+php库模块。

ngx_http_fastcgi_module和ngx_http_proxy_module很像，在proxy里的用的proxy_pass，在fastcgi里就变成了fastcgi_pass，定义缓存也换成了是fastcgi_cache_path，调用缓存也换成了fastcgi_cache等等。基本用法都差不多。
