位置变量

位置变量：在脚本代码中调用通过命令行传递给脚本的参数

  • $1, $2, ...$9, ${10}：对应脚本后参数的位置，超过两位数后要加花括号。
  • $0: 脚本本身（绝对路径）

结果：

  • $*: 传递给脚本的所有参数，全部参数合为一个字符串
  • $@: 传递给脚本的所有参数，每个参数为独立字符串
  • $#: 传递给脚本的参数的个数

tips: $@ $* 只在被双引号包起来的时候才会有差异

  • shift NUM可以用来想做移动位置参数。

退出状态

0代表命令运行成功
1-255代表命令运行失败

脚本中可以自定义退出码，比如exit 100
$?上一个命令运行时候的退出状态码。

&& 与 ||

&&：前面命令成功，则运行后面命令。
||：前面命令失败，则运行后面命令。
